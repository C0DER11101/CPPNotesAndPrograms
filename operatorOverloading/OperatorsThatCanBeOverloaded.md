# Operators that can be overloaded

| operator |    Description               |
|  ------  |        -------------         |
| sizeof   | Size of operator             |
|  ------  |        -------------         |
| .        | Membership operator          |
|  ------  |        -------------         |
| .*       | Pointer-member operator      |
|  ------  |        -------------         |
| ::       | Scope resolution operator    |
|  ------  |        -------------         |
| ?:       | Conditional operator         |
